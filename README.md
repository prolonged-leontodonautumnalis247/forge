## 🧭 What is forge?

forge is a Windows app that helps you run and manage groups of AI agents from one place. It is built for people who want to set up agent work, keep each part aligned, and watch progress in real time.

It brings three core parts together:

- Auto-design for agent swarms
- Contract checks so parts fit together
- A live dashboard you can use while tasks run

This makes it easier to set up multi-agent work without handling each piece on its own.

## 🧩 How forge works

forge is built around a simple flow:

1. You give it a goal
2. It builds a group of agents around that goal
3. Each agent gets a clear role
4. Contracts check that each part matches the next part
5. The dashboard shows what is happening in real time

This helps avoid broken handoffs and makes it easier to follow the work from start to finish.
